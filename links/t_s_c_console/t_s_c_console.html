Travelport Search Control Console

Инструмент Search Control Console компании Travelport – единственное в своем роде интернет-решение, которое позволяет агентству настраивать параметры поиска авиаперелета под потребности клиента. Например, агент сможет удалить из поиска определенные транзитные аэропорты, с которыми у клиента связаны неприятные воспоминания, либо временно заблокировать в поиске стыковочный аэропорт, который находится в регионе, подвергшемся стихийному бедствию. Новый инструмент позволит устранить необходимость трудозатратного кодирования и отправления инструкций кассирам, что позволит сэкономить время и сократит издержки.

Данное решение позволит индивидуализировать поиск в реальном времени. Агентства смогут управлять более чем 35 параметрами и настраивать поиск под конкретные пожелания клиента.

Search Control Console осуществляет эффективное управление результатами поиска благодаря набору ограничителей поиска, которые позволяют:

  • исключить ночные перелеты;
  • установить ограничение для длительности пересадки;
  • установить максимальное число пересадок (в зависимости от перевозчика);
  • запретить выдачу вариантов стыковок в разных аэропортов;
  • ограничить трансфер между перевозчиками;
  • сортировать результаты поиска по типу тарифа, семейству тарифов, коду базового тарифа;
  • устанавливать приоритет для прямых перелетов (без учета стоимости);
  • устанавливать ограничения по номерам рейсов или по их диапазону.

Инструмент Search Control Console может также помочь агентствам в достижении важных финансовых показателей. Поиск можно настроить таким образом, чтобы выполнять план продаж по конкретным авиакомпаниям.
