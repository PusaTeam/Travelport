Разработчикам ПО

Откройте для себя новые возможности — присоединяйтесь к Travelport Developer Network.

Travelport Developer Network — это специализированный портал, который обеспечивает доступ к широкому спектру ресурсов и служб поддержки.

Преимущества Travelport Developer Network:

  • Поиск дешевых авиатарифов перед бронированием или изменением маршрута
  • Поиск вариантов перелета, доступных по определенной цене, с учетом акций авиакомпаний
  • Выделенная команда поддержки со стороны Travelport – закрепляются специалисты технической и коммерческой служб.
  • Приоритетный процесс прохождения сертификации разработки на базе uAPI.
  • Уникальная рекламная площадка Travelport Developer Directory для продвижения своих продуктов целевой аудитории, насчитывающей 67000 агентств-пользователей Travelport GDS в 170 странах мира.
  • Единый доступ к ресурсам для разработки, таким как схемы и документации, файлы справок, демонстрации, информационные бюллетени, файлы загрузок, брошюры.
  • Возможность участия в отраслевых мероприятиях, конференциях, форумах, включая Travelport Network Developer Forum, для обмена опытом и новостями с ведущими мировыми IT компаниями.

Благодаря Travelport Developer Network у компаний занимающихся разработкой программных решений появляется больше возможностей для продвижения своего продукта целевой аудитории.
